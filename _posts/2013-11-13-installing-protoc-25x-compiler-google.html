Using Homebrew
      
     brew install protobuf


Building from source

  1. Download latest version of procbuffer. [https://code.google.com/p/protobuf/downloads/list].
    • e.g  "wget https://protobuf.googlecode.com/files/protobuf-2.5.0.tar.bz2"
  2. Untar the tar.bz2 file
    • tar xfvj protobuf-2.5.0.tar.bz2
  3. Configure the protobuf.
    •  cd protobuf-2.5.0
    • ./configure CC=clang CXX=clang++ CXXFLAGS='-std=c++11 -stdlib=libc++ -O3 -g' LDFLAGS='-stdlib=libc++' LIBS="-lc++ -lc++abi"
  4. Make the source 
    • make -j 4 
    • sudo make install

If you have already installed protobuf using Homebrew then uninstall to avoid conflict.
